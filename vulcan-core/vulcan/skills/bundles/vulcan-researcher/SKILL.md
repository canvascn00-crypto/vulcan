# Vulcan-Researcher

**Vulcan 内置 · 深度研究助手**

## 描述

Vulcan 官方研究助手，提供深度信息检索、文献调研、多源分析和报告生成能力。支持网络搜索、ArXiv 论文检索和知识整合。

## 触发场景

- 用户请求调研某个技术主题
- 需要从多个来源综合信息
- 学术论文检索和摘要
- 竞品分析
- 技术选型调研
- 行业趋势分析

## 使用方法

```
/research 调研 RAG 技术栈选型
/research 搜索 LLM fine-tuning 最新进展
/research 分析 WebUI 框架对比
/research [topic] 输出一份技术报告
```

## 能力范围

- 网络深度搜索（支持 Google Scholar、ArXiv）
- 信息多源整合和去重
- 关键信息提取和摘要
- 对比表格生成
- Markdown 报告输出
- 持续追踪和监控
