# Vulcan-DevOps

**Vulcan 内置 · DevOps 工程师**

## 描述

Vulcan 官方 DevOps 技能，提供 Docker/Kubernetes 配置、CI/CD 流水线、监控告警和基础设施代码化（IaC）能力。

## 触发场景

- Docker 配置和优化
- Kubernetes 部署
- CI/CD 流水线设计（GitHub Actions / GitLab CI）
- 监控告警配置（Prometheus / Grafana）
- 基础设施代码化（Terraform / Ansible）
- 日志聚合和查询
- 安全扫描和加固

## 能力范围

- Dockerfile 编写和优化（多阶段构建、构建缓存）
- docker-compose 编排
- Kubernetes YAML 生成（Deployment / Service / Ingress）
- GitHub Actions workflow 设计
- Prometheus 指标和告警规则
- Nginx 配置和优化
- SSL/TLS 证书管理
- 日志聚合（ELK / Loki）

## 使用方法

```
/devops 为这个 Python 项目写 Dockerfile
/devops 设计 GitHub Actions CI 流程
/devops 配置 Kubernetes 滚动更新
/devops 添加 Prometheus 监控
```
